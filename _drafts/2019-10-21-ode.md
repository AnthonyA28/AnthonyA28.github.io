## A Few Quick Examples.

Lets solve some ordinary differential equations numerically with python. 

### Example 1

Here is a simple example to start with.

This is pretty simple to do in your head.  It is ![1571709536265](\assets\images\ode\1571709536265.png). If we are given an initial value for this, it can be solved numerically very simply. The definition of a derivative says that for a small enough time interval, the derivative can be approximated as: ![1571709727213](\assets\images\ode\1571709727213.png).  Isolate  y₂ to get ![1571710198711](\assets\images\ode\1571710198711.png). In our case, we are integrating with respect to x instead of t. Define the constant h as ![1571710360619](\assets\images\ode\1571710360619.png) which is our step size, perhaps 0.5 . At yₒ is the initial condition, lets say it is 4. Not lets see how this would look in Python as we go from n = 0 to n = 10.

```python
h = 0.5		# time n interval
y_init = 4.0	# initial condition on y 
N = 10	# number of steps

import numpy as np
import matplotlib.pyplot as plt
y = np.zeros(N)	
y[0] = y_init
x = np.zeros(N)

def dy_dx(x, y):	# RHS of the differential
    return x

_x = 0.0	# initial step is at x=0 
for n in range(1,N):	# g through each step n = [1,9]
    y[n] = y[n-1] + h * dy_dx(x[n-1],y[n-1])
    _x = _x + h; 
    x[n] = _x;

plt.plot(x, y) 
plt.xlabel('x')
plt.ylabel('y')
plt.show()
print(y)
print(x)

```
